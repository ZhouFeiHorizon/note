登录码云 

https://gitee.com/

之后新建仓库

![1641730788756](README_files/同步到码云/1641730788756.png)



输入仓库信息

![1641730845979](README_files/同步到码云/1641730845979.png)

点击创建

创建成功了之后会有下面这个界面

![1641730959144](README_files/同步到码云/1641730959144.png)

拷贝第一个箭头的那个, 拷贝仓库地址 `https://gitee.com/zh8739/qkl.git`

然后回到项目中（我们公司的项目）执行命令

```shell
git remote set-url --add origin https://gitee.com/zh8739/qkl.git

git push origin --all
```



![1641732525050](README_files/同步到码云/1641732525050.png)
